# TaskApi

Atividade para por em prática os conhecimento em C#, .NET e suas bibliotecas.

Bibliotecas utilizadas:
	-Entity Framework Core;
	-AutoMapper;
	-Pomelo Entity FrameworkCore MySql.

A API consiste num sistema de gerenciamento de tarefas. As tarefas são compostas pelo seguinte:
	-Título;
	-Descrição;
	-Data de início;
	-Lista de atividades;
	-Lista de categorias;
	-Status.

As atividades são compostas pelo seguinte:
	-Título;
	-Descrição;
	-Data de início;
	-Data de conclusão;
	-Tempo (em horas) para conclusão;
	-Status.

	As informações são integradas ao banco de dados SQL e recebidas por meio do protocolo HTTP.
